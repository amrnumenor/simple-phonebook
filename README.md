## About The Project

Simple create-and-read web app that adds your friends' contact and view them on the site.

### Built With

* [PHP](https://www.php.net/)
* [MySQL](https://www.mysql.com/)

<!-- GETTING STARTED -->
## Getting Started

Set up MAMP and virtual host

### Prerequisites

1. PHP Environment
2. [MAMP](https://www.mamp.info/en/windows/)
